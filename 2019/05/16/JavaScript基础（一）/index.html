##javascript基础（一）

###对象

####对象属性
为了实现属性特性的查询和设置操作，es5中定义了4个属性，分别为value,writable,enumerable,configurable

  • 获取属性特性

Object.getOwnPropertyDescriptor(某个对象，属性) 可以获取某个对象特定属性的属性描述符，但是只能获取自有属性的描述符

  • 设置属性特性

    Object.defineProperty(对象，对象属性，{value:1,writable:true,
    enumerable:false,configrable:true}

原型属性

检测一个对象是否是另一个对象的原型（或处于原型链中），
x.isPrototypeOf(y)

####对象序列化
JSON.stringify() JSON.parse()

数组

var a = [2,1,3]

  • a.join(“ “) =>”2 1 3”
  • a.reverse() => [3,1,2] 改变原数组的值

  • a.sort() 按字母表进行排序 =>[1,2,3]

  • a.concat(4,5) =>[2,1,3,4,5] 返回新数组

  • a.slice(1,3) =>从包含的起始位置到不含第二个参数指定的位置的数组
  • a.splice(开始位置，删除元素个数，新增元素) 修改原数组
  • a.push() 尾部添加元素，并返回数组长度
  • a.pop() 删除最后一个元素，并返回删除的值
  • a.unshift() 头部添加元素，返回数组长度
  • a.shift() 头部删除元素，并返回删除的值
  • a.toString() 将每个元素转为字符串
  • a.forEach(function(数组元素，索引，数组本身){//不能使用break等中断语句})
  • a.map(function(value,index,arr){}). 返回新的数组
  • a.filter(function(value,index,arr){}). 满足true 的元素返回
  • a.every(function(v,i,arr){}) 所有元素都满足返回true =>true
  • a.some(function(v,i,arr){}). 至少一个为true的返回
  • a.reduce(函数，初始值(可选)).函数指的是化简操作的函数
  • a.reduceRight() 用法同上,按照索引从高到低
  • a.indexOf(). 给定值的索引
  • a.lastIndexOf(). 反向索引
  • Array.isArray(a). 判断是否为数组
