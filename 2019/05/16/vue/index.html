Vue

###vue-router

####跳转

  • 组件 ，默认渲染为a标签,to指定跳转的路径
    <router-link to="/about">跳转到about</router-link>

router-link常用属性:

  1. tag 渲染标签
  2. active-class 匹配成功样式
  • 在js中跳转 this.$router.push(‘about’)

属性：
1.replace 替换当前的history记录
2.go 在history中向前或向后退几步

  • beforeEach(to(即将进入的目标的路由对象),from(当前导航即将要离开的路由对象),next（调用该方法，才能进入下一个钩子）) afterEach()
  • next(参数)，设置具体的路径可以导航到指定的页面

###Vuex(统一管理组件状态)

  • 例如在store的state状态中定义一个数据count，初始化为0，就可以在其他组件中通过$store.state.count获取
  • 改变count的值，需要显示的提交mutations,在组件内通过this.$store.commit(‘increment’)来执行mutations.改变count的值
  • mutations中尽量不能异步操作数据，组件在commit后，数据不能立即改变
  • 过滤元素，用getters方法，在组件的computed方法中通过this.$store.getters.filteredList调用，同时getter也可以依赖其他的getter
  • 异步请求数据，在actions中做处理，通过组册方法this.$store.dispatch(‘increment’)调用
  • modules 用来将store分割为不同的模块
